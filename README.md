# Cable Customer Management System

A full-stack web application designed to digitize customer records and payment tracking for a local cable service provider. This system replaces traditional manual record-keeping with a structured and efficient digital solution.

---

## Features

- Customer Management  
  Add, update, and manage customer details

- Payment Tracking  
  Record and monitor customer payments

- Basic Reporting  
  View customer status and payment history

- Search and Filter  
  Quickly find customers using search functionality

- Responsive UI  
  Works across desktop and mobile devices

---

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

---

## Project Overview

This project was built to solve a real-world problem faced by a local cable operator who maintained customer records manually. The application provides a centralized system to manage customer data and track payments efficiently.

---

## Installation and Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

#### Backend:

``` bash
cd server
npm install
```

#### Frontend:

```bash
cd client
npm install
```


### 3. Configure Environment Variables

##### Create a .env file in the server directory:

MONGO_URI=your_mongodb_connection_string
PORT=5000

### 4. Run the application

##### Start backend:

```bash
cd server
npm start
```
#####  Start frontend:

```bash
cd client
npm run dev
```
